O que é HTML?

HTML é uma Linguagem de Marcação de Hipertexto, e é usada na criação de páginas web, essa linguagem foi desenvolvida pelo inglês Tim Berners-Lee, e a primeira versão do HTML foi criada em 1993.

Histórico de versões do HTML:

  1. HTML (1991): A primeira versão, criada por Tim Berners-Lee, era um conjunto limitado de tags para compartilhar documentos científicos.

  2. HTML 2.0 (1995): A primeira versão padronizada, com características principais como tabelas e formulários.

  3. HTML 3.2 (1997): Introduziu novas tags e recursos, incluindo suporte para tabelas mais complexas e a possibilidade de incluir imagens.

  4. HTML 4.01 (1999): Versão que trouxe melhorias significativas, como suporte a CSS e scripts, além de melhorar a acessibilidade e a internacionalização.

  5. XHTML 1.0 (2000): Uma versão do HTML que segue as regras do XML, buscando uma maior compatibilidade com outras tecnologias web.

  6. HTML5 (2014): A versão mais recente, que trouxe novas funcionalidades como suporte a multimídia (áudio e vídeo), novas tags semânticas, e APIs para desenvolvimento web. O HTML5 foi desenvolvido a partir de 2004 pela WHATWG.

Tim Berners-Lee e a Web

Algumas frases que o criador do HTML declarou para a web

⁠O poder da Web está em sua universalidade. Acessibilidade para todos independente de incapacidade é um aspecto essencial.
⁠O objetivo da Web é servir à humanidade. Nós a construímos agora para que aqueles que venham a ela mais tarde possam criar coisas que nós mesmos não conseguimos imaginar.
A web não está concluída, é apenas a ponta do iceberg. As novas mudanças irão balançar o mundo ainda mais.
